# Katkıda Bulunma

1. Fork'la
2. Yeni branch aç (`git checkout -b yeni-ekleme`)
3. Değişiklikleri yap
4. Commit et (`git commit -am 'Yeni kaynak eklendi: ...'`)
5. Push et (`git push origin yeni-ekleme`)
6. PR aç

## Format

Tablo formatına uygun ekleme yapın:

```markdown
| İsim | Açıklama | Link |
|---|---|---|
```
